# chain-game
simple browser game like cell automaton.

## how to play
visit [here](https://MICKeyzwo.github.io/chain-game)

## how to develop
1. Install `parcel-bundler` with global flag: `npm i -g parcel-bundler`
1. Install dependencies: `npm i`
1. run `npm start`

## how to build
1. Install dependencies: `npm i`
1. run `npm run build`
